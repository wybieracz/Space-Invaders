# Space-Invaders
Implementation of Space Invaders, classic 2D game from 1978. 
## General information
* Classic 2D game
* Three types of enemies and UFO
* FullHD textures
* Sound effects
* Best score saved
## Technologies used
* C++
* SFML
## Controls
* ←→ - move left and right
* Space - shoot
* Enter - start the game
## Requirements
* Processor x86-64
* Windows 7, 8, 10, 11
* Monitor with resolution 1920x1080
* Keyboard
